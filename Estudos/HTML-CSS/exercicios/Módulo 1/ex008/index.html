Trechos de código

Como programadores, acabamos querendo compartilhar nossos códigos, seja na linguagem que for. Por isso, a tag <code> foi criada, informando ao navegador que é um código de computador. Além do efeito para o navegador, há um efeito visual para o usuário, com as letras mono-espaçadas, o que facilita a leitura.

for nota in notas: soma_das_notas += nota print(soma_das_notas)

Provavelmente você percebeu que esse código não tá tão "código" assim. Pra manter a formatação do código, é so usar a <code> dentro da tag <pre>, que representa texto pré-formatado. 

        
            for nota in notas:
            soma_das_notas += nota
