Principais formatações

Negrito

Existem duas formas de de colocar-se negrito:

Usando <b> como nessa palavra.

Ou utilizando <strong> como nessa palavra.

"Mas qual a diferença?". Utilizando <b>, você apenas alterou a forma, sem modificar o significado, a semântica. Já com <strong>, os termos nele estão em destaque semanticamente.

Itálico

Além do negrito, também temos o itálico, seguindo a mesma divisão de ter ou não valor semântico:

Sem valor, usando <i>:
Fancy

Com valor, usando <EM>:
Fancy, but ✨spicy✨

Texto marcado

Outro tipo de texto possível, é o texto marcado, utilizando a tag <mark>.

"Nossa, mas só tem essa cor? Não combina nada com as cores do meu site."
Bom, com HTML não tem como ser outra. Mas utilizando CSS, o céu é o limite ^^

Com o parâmetro style, é possível mudar a cor com o background-color!

Olha que lindo que fica azul S2

Pra não precisar mudar todos os mark um por um, você pode fazer isso com o style no head.

Texto grande e pequeno

Outra coisa possível é diminuir o tamanho das letras com <small>. Também era possível aumentar utilizando <big>, mas essa tag é obsoleta. Se quiser configurar o tamanho das letras com mais precisão, é inevitável o uso de CSS.

Texto deletado e inserido

Pra sinalizar a exclusão ou inserimento de um novo texto. São usada as tags <del> e <ins>.

Esse aqui foi de arrasta pra cima

Esse acabou de chegar

Para utilizar esses textos sem semântica, é necessário configurar através do CSS, com text-decoration.

Texto sobrescrito e subscrito

Sabe aqueles números pequenos que ficam em cima ou embaixo, como em elementos químicos na tabela periódica? Tem como colocá-los! Com as tags <sub> e <sup>, o texto fica assim:

H2O e x2

Trechos de código

Como programadores, acabamos querendo compartilhar nossos códigos, seja na linguagem que for. Por isso, a tag <code> foi criada, informando ao navegador que é um código de computador. Além do efeito para o navegador, há um efeito visual para o usuário, com as letras mono-espaçadas, o que facilita a leitura. Pra exemplificar melhor, vou usar um pouco do código usado nessa página.

document.getElementById('demo').innerHTML = 'Hello JavaScript';

Citações

Sabe quando você escreve um texto e usa uma frase de outra pessoa, uma citação, para explicar o seu argumento? Em HTML, esse tipo de texto é feito a partir da tag <q>.

Só sei que nada sei - Alguém que definitavemente não foi Sócrates.

Com citações mais longas, é usado <blockquote>, que cria um parágrafo para si. E se quiser colocar um link para o texto é só usar o parâmetro cite dentro da tag.

Abreviações

Já no casos das abrevições, como HTML (Hyper Text Markup Language), há momentos em que você quer deixar claro tanto para o usuário quanto para o mecanismo de busca, o significado da mesma. Com o <abbr>, além de dar esse valor, também mostra a significado dela ao passar o mouse por cima.

Texto oditrevni

Pra ter um texto como esse, é só usar a tag <bdo> (bi-directional override). Pra definir a direção, coloque o parâmetro dir e as variações possiveis <rtl> (direita pra esquerda) e ltr (esquerda pra direita) dentro da tag.
