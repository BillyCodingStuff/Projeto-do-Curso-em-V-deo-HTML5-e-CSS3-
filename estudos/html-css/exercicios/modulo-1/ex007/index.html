Principais formatações

Negrito

Existem duas formas de de colocar-se negrito:

Usando <b> como nessa palavra.

Ou utilizando <strong> como nessa palavra.

"Mas qual a diferença?". Utilizando <b>, você apenas alterou a forma, sem modificar o significado, a semântica. Já com <strong>, os termos nele estão em destaque semanticamente.

Itálico

Além do negrito, também temos o itálico, seguindo a mesma divisão de ter ou não valor semântico:

Sem valor, usando <i>:
Fancy

Com valor, usando <EM>:
Fancy, but ✨spicy✨

Texto marcado

Outro tipo de texto possível, é o texto marcado, utilizando a tag <mark>.

"Nossa, mas só tem essa cor? Não combina nada com as cores do meu site."
Bom, com HTML não tem como ser outra. Mas utilizando CSS, o céu é o limite ^^

Com o parâmetro style, é possível mudar a cor com o background-color!

Olha que lindo que fica azul S2

Pra não precisar mudar todos os mark um por um, você pode fazer isso com o style no head.

Texto grande e pequeno

Outra coisa possível é diminuir o tamanho das letras com <small>. Também era possível aumentar utilizando <big>, mas essa tag é obsoleta. Se quiser configurar o tamanho das letras com mais precisão, é inevitável o uso de CSS.

Texto deletado e inserido

Pra sinalizar a exclusão ou inserimento de um novo texto. São usada as tags <del> e <ins>.

Esse aqui foi de arrasta pra cima

Esse acabou de chegar

Para utilizar esses textos sem semântica, é necessário configurar através do CSS, com text-decoration.
